### 万字文档+论坛管理系统+Vue2+SpringBoot
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、VSCode
技术栈：Java、MySQL、SpringBoot、Mybatis/MybatisPlus、Vue2、ElementUI、JavaScript、CSS、node.js14、HTML、Laiyui

系统功能模块：前台使用HTML+Laiyui，后台使用vue+ElementUI
角色有3种，分别是游客、会员和管理员
管理员：个人中心、版主管理、用户管理、留言版管理、个人中心、版主管理、新闻信息管理、轮播图管理、基础数据管理
版主：论坛管理、新闻信息查看、个人信息
用户：留言板、在线论坛、新闻信息
